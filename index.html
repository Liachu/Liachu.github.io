Mes Projets

Création d'un jeu massivement multijoueur d'action en bourse (En cours) (HTML/CSS/JS)

Le Rat de Wall Street est un party game multijoueur jouable sur téléphone, avec un écran commun projeté pour tous les participants. Dans ce jeu, la réactivité est la clé du succès. L'objectif était de créer une expérience multijoueur où chaque utilisateur peut spéculer sur un cours boursier dynamique, géré par un moteur algorithmique côté serveur. Développée avec Node.js et Express, l'architecture repose sur la puissance des WebSockets via Socket.io pour garantir une latence minimale : chaque variation du marché et chaque transaction est diffusée instantanément à l'ensemble des participants. L'interface utilise Chart.js pour une visualisation graphique des données. Ce jeu est en cours de développement. Je conseille fortement d’aller voir le GDD (Game Design Document) dans le dépôt GITHUB ci-dessous.

HTML5 CSS JavaScript Express.js Socket.io Architecture Client-Serveur Cloud Hosting Railway UML Modeling UX / UI Design

Pixel Quest, Jeu vidéo 2D à niveaux (C++)

Pixel Quest est un jeu vidéo 2D en pixel art où le joueur incarne un personnage coincé dans une mystérieuse demeure hantée. Son objectif : éviter les esprits errants et survivre le plus longtemps possible en se faufilant habilement entre les obstacles et en atteignant les portes de chaque niveau pour atteindre le suivant. Les pièces dispersées dans la pièce sont des bonus. Il faut les récupérer en esquivant les ennemis. Ce jeu a été réalisé avec une direction artistique rétro et une ambiance immersive
